shipping README
==================

Getting Started
---------------

- cd <directory containing this file>

- git submodule init

- git submodule update

- cd pyactiveresource && python setup.py install && cd ..

- cd shopify_python_api && python setup.py develop && cd ..

- $VENV/bin/python setup.py develop

- $VENV/bin/initialize_shipping_db development.ini

- $VENV/bin/pserve development.ini
